# Iniciando com ASP.NET CORE 2.0

## ASP.NET CORE CLI

-dotnet new
-dotnet run
-dotnet build
-dotnet restore
-dotnet publish
-dotnet sln
-dotnet add package
-dotnet add reference

## Etapas do projeto

1.Criar o projeto MVC com autenticação individual e segurança
  dotnet new mvc --auth individual -o Tarefas

2.Entrar na pasta
  cd Tarefas

3.Executar
  dotnet run
